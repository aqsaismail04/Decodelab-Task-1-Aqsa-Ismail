DecodeLabs Python Internship — Project 1 | Batch 2026

---

## Overview

A command-line To-Do List application built with Python.
Users can add tasks, view tasks, and data is saved to a file.

---

## Features

- Add new tasks
- View all saved tasks with numbering
- Tasks saved to file — data persists after closing
- Empty task validation
- Total task count display
- Graceful exit with goodbye message

---

## Concepts Used

- Python Lists and append()
- while loop and for loop
- if / elif / else
- enumerate() for numbered list
- try / except for error handling
- File Handling for saving tasks
- Functions for clean code

---

## How to Run

1. Clone the repo
2. Go to Project-1 folder
3. Run this command:

python todolist.py
