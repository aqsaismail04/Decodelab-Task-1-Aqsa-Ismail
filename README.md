🐍 Python To-Do List Application (CLI + File Handling)

Developed a Python command-line To-Do List application with file handling functionality, enabling users to add, view, and manage tasks with persistent data storage.

🚀 Overview

A simple yet powerful command-line To-Do List application built using Python.
It allows users to manage daily tasks efficiently, with all data saved permanently using file handling.

✨ Features

✔ Add new tasks
✔ View all saved tasks (with numbering)
✔ Persistent storage using file handling
✔ Automatic loading of previous tasks
✔ Input validation (empty task protection)
✔ Total task count display
✔ Clean exit with goodbye message

🧠 Concepts Used
Python Lists (append())
Loops (while, for)
Conditional Statements (if / elif / else)
enumerate() for numbering
Functions for modular code
File Handling (read/write tasks)
Exception Handling (try / except)
