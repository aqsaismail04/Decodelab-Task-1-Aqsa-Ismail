🐍 Python To-Do List Application (CLI + File Handling)










🚀 Developed a Python command-line To-Do List application with file handling functionality, enabling users to add, view, and manage tasks with persistent data storage.


📌 Overview


A simple yet powerful command-line To-Do List application built using Python.
It helps users manage daily tasks efficiently with permanent file-based storage, ensuring data is never lost after closing the program.


✨ Features


✔ Add new tasks
✔ View saved tasks with numbering
✔ Persistent storage using file handling
✔ Auto-load previous tasks on startup
✔ Input validation (empty task protection)
✔ Total task counter
✔ Clean and user-friendly CLI interface
✔ Graceful exit system


🧠 Concepts Used


Python Lists (append())
Loops (while, for)
Conditional Statements (if / elif / else)
enumerate() for indexing
Functions for modular programming
File Handling (read/write operations)
Exception Handling (try / except)
